# Project Eulerを行うためのプリセット
## 手順
解答のコードを書くためのファイルを作成する eulerXXX.rb ( XXX には問題の数字が入る。例: euler001.rb )


```
rake all           # 全員分を実行する
rake daily_report  # １日分のレポートを保存する
rake default       # 自分の分を実行する
rake report        # レポートを出力する
rake update        # 全員分のリポジトリをアップデートする
rake upload        # ファイルをアップロードする
```
